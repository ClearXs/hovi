# 评分标准应对指南

## 评分办法类型

### 综合评分法

综合评分法是最常见的评标办法，总分由技术标得分和商务标得分加权计算。

**计算公式**：

```
总分 = 技术标得分 × 技术权重 + 商务标得分 × 商务权重
```

**常见权重分配**：

- 技术60% + 商务40%
- 技术70% + 商务30%
- 技术50% + 商务50%

### 最低价法

以价格最低为中标条件，技术标通过性审查。

### 经评审的最低价法

在技术标通过的前提下，价格最低者中标。

## 常见评分因素及应对策略

### 一、技术评分因素

#### 1. 技术方案 (15-30分)

**评分要点**：

- 方案的完整性

- 方案的可行性
- - 方案的先进性方案的针对性

**应对策略**：

- 详细描述技术路线
- 突出创新点和优势
- 提供详细的技术架构图
- 结合项目实际需求

#### 2. 项目实施 (10-20分)

**评分要点**：

- 实施计划的合理性
- 进度安排的可行性
- 风险控制措施
- 里程碑设置

**应对策略**：

- 制作详细的实施计划表
- 明确各阶段交付物
- 制定风险应对预案
- 预留合理的缓冲时间

#### 3. 质量保证 (5-10分)

**评分要点**：

- 质量管理体系
- 测试方案
- 缺陷处理流程

**应对策略**：

- 展示ISO/CMMI等认证
- 详细说明测试流程
- 提供质量指标承诺

#### 4. 售后服务 (5-10分)

**评分要点**：

- 响应时间
- 维护方案
- 培训计划

**应对策略**：

- 明确响应时间承诺
- 详细的维护方案
- 完善的培训计划

#### 5. 项目团队 (10-20分)

**评分要点**：

- 团队规模
- 人员资质
- 类似项目经验

**应对策略**：

- 突出核心成员资质
- 展示类似项目业绩
- 提供团队简历

#### 6. 企业实力 (5-15分)

**评分要点**：

- 注册资金
- 营业收入
- 人员规模
- 资质等级

**应对策略**：

- 突出企业优势
- 展示各类资质证书
- 提供业绩数据

---

### 二、商务评分因素

#### 1. 投标报价 (20-40分)

**评分计算方式**：

| 方式       | 计算公式                            |
| ---------- | ----------------------------------- |
| 低价优先法 | 得分 = (最低价/投标价) × 权重       |
| 基准价法   | 得分 = 权重 × (1 - \|偏差\|/基准价) |
| 加权平均法 | 得分 = 权重 × (1 - 偏差率)          |

**应对策略**：

- 合理报价，避免过高或过低
- 仔细研究预算和限价
- 考虑成本和利润平衡

#### 2. 财务状况 (5-10分)

**评分要点**：

- 资产负债率
- 流动比率
- 营业收入
- 利润情况

**应对策略**：

- 提供审计报告
- 展示良好的财务指标

#### 3. 业绩 (10-20分)

**评分要点**：

- 类似项目数量
- 合同金额
- 用户满意度

**应对策略**：

- 整理类似项目业绩
- 提供合同复印件
- 获取用户评价

## 得分策略建议

### 1. 关键得分点识别

- 优先保障高权重项目得分
- 避免明显失分项

### 2. 差异化竞争

- 突出自身优势
- 展示独特价值

### 3. 响应针对性

- 逐条响应招标要求
- 避免遗漏评分点

### 4. 证据支撑

- 用数据说话
- 用案例证明
- 用证书背书
